# Alpaca Desktop

[BitBar][1] distribution including [Alpaca][2] and [launchd][3] daemon and [BitBar][1] plugin.

![Screenshot](docs/screenshot.png)

## Install

`Alpaca.app` can be downloaded from [here][4].

## Build

```
bin/build
```

Creates a `dist/Alpaca.app`.
